# ChurnGaurd-Ai
Customer Churn Prediction &amp; Segmentation System End-to-end ML project predicting customer churn with 82% accuracy. Features customer segmentation (K-Means), CLV calculation, and interactive Streamlit dashboard. Identified 4 customer segments with churn rates from 4% to 46%. Built with Python, Scikit-learn, XGBoost, and Streamlit.
